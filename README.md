# imagine-store
Site de e-commerce. Consome uma API responsável por ceder todas as informações de produtos e guardar informações de usuários do site. Possui login, autenticações, filtros de produtos, favoritos... infelizmente trâmites de compras ainda não foram finalizados.

Embora a moeda corrente escolhida foi o Real, o idioma do site está todo em inglês devido aos dados da API serem fornecidos neste idioma. O nome do projeto surgiu devido a inconsistência dos dados trazidos pela API, onde fotos e nomes são totalmente aleatórios, restando apenas imaginar os "produtos" ao navegar no site. rsrs

## Techs envolvidas:
- _**React**_ + _**Typescript**_ para desenvolver resultados mais consistentes
- _**React Router Dom**_ para criar rotas o mais dinâmicas possíveis
- _**Framework React Bootstrap**_ para acelerar todo o processo de estilização e montagem de componentes genéricos
- _**Scss (Sass)**_ por agilizar a disposição das precedencias de cada elemnto através de encapsulamento
- _**Eslint**_, uma ajuda indispensável para criar códigos mais organizados e padronizados
- _**JS Cookie**_ para lidar com todas as funcionalidades que envolvem Cookies
- _**JWT Decode**_ devido à incompatibilidades do JsonWebToken padrão com o React
- _**Axios**_, que traz uma forma bem mais completa e suscinta de fazer fetch
- _**React Icons**_, todos os ícones que preciso em um só lugar.

## Como rodar:
Primeiramente, clone o repositório deste projeto.

Após isso, basta instalar as dependências necessárias através do comando `npm install`, e após concuído tal, será possível rodar o ambiente de desenvolvimento através do comando `npm start`.
